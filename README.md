PYTHON SEARCH ENGINE

Description

This project is a simple local search engine built in pure Python. It reads multiple text files, processes their content, and creates an inverted index to allow fast and efficient keyword searches. Users can enter one or more keywords, and the system returns the most relevant documents based on the search query.

The purpose of this project is to demonstrate core computer science concepts such as data structures, algorithms, and text processing. I built it to strengthen my understanding of how search engines work internally, especially indexing and information retrieval. It also reflects my ability to design clean, modular, and efficient Python applications without relying on external frameworks.

The technologies which are used : Python and JSON.

How to run: python main.py
